# News Junction Mobile (Phase 6A)

Standalone Expo + React Native foundation connected to the existing Firebase project.

## Quick start

1. Copy `.env.example` to `.env`
2. Install dependencies: `npm install`
3. Start dev server: `npm run start`
4. Run Android/iOS with `npm run android` / `npm run ios`

## Scope of this phase

- App foundation, architecture, navigation, onboarding, auth baseline, settings, offline and notification foundation.
- No news feed logic, AI logic, comments, analytics, ad/revenue, or bookmark/search business logic.

## Notes

- App is intentionally separate from web code.
- Uses same Firebase project credentials.
- Google Sign-In is scaffolded as placeholder; full Expo AuthSession integration is planned for Phase 6B.
