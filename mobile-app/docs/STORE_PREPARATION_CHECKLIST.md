# Store Preparation Checklist

## Google Play readiness
- [ ] Adaptive icon, splash assets finalized
- [ ] Android package id verified
- [ ] Notification permission rationale documented
- [ ] Data safety form prepared
- [ ] Testing tracks configured (internal/closed/open)

## Apple App Store readiness
- [ ] iOS bundle id verified
- [ ] App privacy details prepared
- [ ] Sign-in and account deletion policy validated
- [ ] iPad screenshots ready
- [ ] Review notes prepared

## Shared assets
- [ ] App icon set
- [ ] Feature graphic
- [ ] Screenshots (reader + admin + AI + diagnostics)
- [ ] Release notes templates
- [ ] Support and privacy URLs
