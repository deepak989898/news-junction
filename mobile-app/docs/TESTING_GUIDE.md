# Testing Guide

## Checklists

### Regression checklist
- [ ] Core navigation works
- [ ] Home feed + article open
- [ ] Search + bookmarks + history
- [ ] AI center actions
- [ ] Admin dashboard and article updates

### Performance checklist
- [ ] Cold launch under expected threshold
- [ ] Screen transitions smooth
- [ ] No severe slow frames on key screens
- [ ] API timings captured in metrics

### Offline checklist
- [ ] Offline banner appears
- [ ] Cached content readable
- [ ] Retry queue captures failed writes
- [ ] Sync recovers after reconnect

### Accessibility checklist
- [ ] Large font readability
- [ ] Screen reader labels
- [ ] Contrast and motion settings

### Store submission checklist
- [ ] Icons/splash assets present
- [ ] Privacy and permissions docs complete
- [ ] Release notes and metadata ready
